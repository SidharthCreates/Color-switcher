# Dynamic Color Switcher 🎨

A lightweight, interactive web application that allows users to change the background color of the page dynamically using JavaScript.

## 🚀 Purpose
This project was built to demonstrate proficiency in DOM manipulation, event listeners, and CSS transitions. It focuses on creating a smooth user experience with clean code.

## 🛠️ Tech Stack
* **HTML5**: Structured semantic layout.
* **CSS3**: Responsive design with smooth background transitions.
* **JavaScript**: Logic to handle button clicks and style updates.

## ✨ Features
* **Real-time Updates**: Background changes instantly without page reload.
* **Smooth Transitions**: Uses CSS `ease` transitions for a premium feel.
* **Clean Code**: Modular JavaScript that is easy to scale.

## 📂 How to Run
1. Clone this repository.
2. Open `index.html` in any modern web browser.
